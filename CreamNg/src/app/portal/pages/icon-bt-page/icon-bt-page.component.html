Properties and Events

Properties applicable to the component app-message-block
Name Type Default Description
aName * string "Action" Accessible name for the button (gives text alternative to icon)
svgId * string "" Id of the svg (should be listed in "/assets/img/sprite.svg")
rLink string "" If a value is given, replaces button by a link and adds value as an absolute URL
disabled string "" Disables button if value is "disabled
discrete (class) -- Icon only style
minimal (class) -- Icon only, smaller; suitable for actions in tables

* mandatory properties
