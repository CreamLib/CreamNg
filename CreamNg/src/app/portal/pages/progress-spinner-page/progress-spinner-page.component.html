General Component

Giving a template variable when calling the component allows you to trigger or stop the spinner.

Please, wait...

 
<app-progress-spinner #process1>
  <p>Please, wait...</p>
</app-progress-spinner>

EXAMPLE TO TRIGGER THE SPINNER

<button (click)="process1.ToggleOpen($event); stopProgress(process1)">Open spinner</button>

Properties and Events

Properties applicable to the component app-progress-spinner
Name Type Default Description
progressText string "Please, wait..." Allows to customize alternative text for screen readers

* mandatory properties
