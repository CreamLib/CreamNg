Content of the modal

  Opens small modal
</app-modal-trigger>
<app-modal class="simple-dialog" #exempleModal2 modalTitle="Cancel contract">
  <p>Content of the modal</p>
  <p class="buttons-bar">
    <button (click)="test();exempleModal2.ToggleOpen()">Ok</button>
  </p>
</app-modal>

If actions inside the modal should close it, you can call the ToggleOpen function using the modal's identifier.

You may add aditional functions to the same event.

Properties and Events

Properties applicable to the component app-modal
Name Type Default Description
modalTitle * string -- Title of the modal
<identifier> * #xxxxx -- Unique identifier of the modal
simple-dialog (class) -- Displays modal in small format, for quick feedback message
Properties applicable to the component app-modal-button
Name Type Default Description
[modal] * string -- Value of the target modal unique identifier
disabled string -- Disables button

* mandatory properties
