Dialog Snackbar

Snackbars work exactly like a modal — in isolated mode —, but they are more discrete, appearing at the bottom of the page.

Use this component to get feedback from the user.

As snackbars impose a question that must be answered, it can only be closed by clicking one of the answers buttons.

Are you sure you want to delete all the files?

 
<app-snackbar #snackbarDemo>
  <NG-CONTENT>
</app-snackbar>

EXAMPLE OF SNACKBAR TRIGGER:

<p><button (click)="snackbarDemo.ToggleOpen($event)">
  Open snackbar
</button></p>

EXAMPLE OF BUTTON ACTION:

<button class="as-reset" (click)="snackbarDemo.ToggleOpen($event)">
  No
</button>
<button class="as-submit" (click)="snackbarDemo.ToggleOpen($event)">
  Yes
</button>

You should use the component's unique identifier (template reference) to call the ToggleOpen function to trigger or close the snackbar.

You may add aditional functions to the same event.
